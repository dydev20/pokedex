# Pokedex

## Description
* A web application that uses the PokeAPI to display Pokemon's from generation 1

## Features 
* Display Pokemon's (name, types and stats) from generation 1
* Modal to show Pokemon stats 
* Search feature to filter Pokemon list

## Technologies
* React
* Material UI
